# Artificial_Intelligence
Para usar esse sistema basta inserir as regras no termianal usando as regras gramaticais impostas. 

- *Gramática*
    - (+ = binary and) 
    - (| = binary or) 
    - (^ = binary xor) 
    - (! = unary not)
    - (=> = implication) 
    - (<=> = bi-diretinal implication) 
    - (=ARGS  = fact list) 
    - (?ARGS = hypothesis list)

- *Execução* 
  - python3 main.py
